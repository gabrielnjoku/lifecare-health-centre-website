A static, single-page website for a fictional community health clinic built using **HTML only** as part of a team web development project. The site demonstrates structured, semantic HTML markup without any CSS or JavaScript.

## Project Structure

The entire website lives in a single file: `project_2_health_clinic.html`

| Section | Description |
|---|---|
| Header & Navigation | Clinic branding and anchor-linked nav menu |
| Home | Clinic introduction, image, and "Why Choose Us" list |
| Services | Definition list of 6 broad services across 2 categories |
| Doctors | Table of 8 staff members with qualifications and availability |
| Appointments | Two-fieldset booking form with all standard input types |
| Health Tips | 3 semantic article elements with ordered tip lists |
| Footer | Address, emergency notice, copyright, and quick links |

## HTML Features Used

- Semantic elements: `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`, `<address>`
- Definition list: `<dl>`, `<dt>`, `<dd>`
- Table with `<caption>`, `<thead>`, `<tbody>`, and `rowspan`
- Form with `<fieldset>`, `<legend>`, `<label>`, `<select>`, `<textarea>`, and input types: `text`, `email`, `tel`, `date`, `time`, `radio`, `checkbox`
- Inline semantics: `<strong>`, `<em>`, `<mark>`

## How to Run

No build tools or dependencies required. Simply open `project_2_health_clinic.html` in any modern web browser.

## Notes

- This project is for educational purposes only. LifeCare Health Centre is a fictional clinic.
- No CSS or JavaScript has been used; all structure and meaning is conveyed through HTML alone.
